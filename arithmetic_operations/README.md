# C Programs

This folder contains a set of simple C programs I've written to practice basic concepts like calculating BMI, working with special characters, and performing mathematical operations. Below is a description of each file along with examples of how they work.

## Files and Descriptions

### 1. `special_characters.c`

This program prints a few special characters to show how C handles them.

#### Example Output

Are all these [*, &, /, , $, %, '] special characters?

### 2. `volume_and_surface.c`

This program calculates the volume and surface area of a cuboid based on the dimensions (length, width, height) provided by the user.

#### Example Input

Enter 3 integers separated by space: 2 3 4

#### Example Output

Volume: 2 _ 3 _ 4 is 24 Surface Area: 2 _ 2 _ 3 + 2 _ 2 _ 4 + 2 _ 3 _ 4 is 52

### 3. `circle_properties.c`

This program calculates the circumference and area of a circle based on the radius provided by the user.

#### Example Input

Enter the radius of the circle (real number): 5.5

#### Example Output

Circle with radius 5.50: circumference = 34.56, area = 95.03

### 4. `average_of_three.c`

This program takes three real numbers from the user and calculates their average.

#### Example Input

Enter 3 real numbers: 4.5 3.3 5.0

#### Example Output

The average of numbers 4.50, 3.30, 5.00 is: 4.267

### 5. `price_with_tax.c`

This program calculates the final price of a product after applying a 20% tax to the user-provided base price.

#### Example Input

Enter the price without tax: 100

#### Example Output

Price without tax: 100 Final price with 20% tax: 120.00

### 6. `bmi_calculator.c`

This program calculates the Body Mass Index (BMI) based on the user's height in centimeters and weight in kilograms.

#### Example Input

Enter your height in centimeters: 175 Enter your weight in kilograms: 70

#### Example Output

BMI: 22.86

## Summary

These programs cover basic C operations and user input/output. They're mostly for practicing handling input, performing arithmetic, and printing results.
